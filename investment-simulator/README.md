# Investment Simulator

Welcome to the Investment Simulator project! This interactive website is designed to teach users about finance and investment strategies through a hands-on simulator experience.

## Project Structure

The project is organized as follows:

```
investment-simulator
├── src
│   ├── index.html          # Main entry point for the website
│   ├── css
│   │   ├── styles.css      # Main styles for the website
│   │   ├── simulator.css    # Styles specific to the investment simulator
│   │   └── responsive.css   # Responsive styles for various devices
│   ├── js
│   │   ├── main.js         # Main JavaScript file for initialization and functionality
│   │   ├── simulator.js     # Logic for the investment simulator
│   │   └── charts.js       # Chart rendering and visualization
│   └── pages
│       ├── dashboard.html   # Dashboard page for investment overview
│       ├── learn.html       # Educational content about finance
│       └── portfolio.html    # Page for managing investment portfolio
├── assets
│   └── fonts
│       └── custom-fonts.css # Custom font styles
└── README.md               # Project documentation
```

## Features

- **Interactive Investment Simulator**: Users can simulate different investment scenarios and see potential outcomes.
- **Educational Content**: Learn about finance and investment strategies through engaging content.
- **Responsive Design**: The website is designed to be fully responsive, ensuring a great experience on all devices.
- **Data Visualization**: Charts and graphs to help visualize investment data and performance.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. For development, you can modify the HTML, CSS, and JavaScript files in the `src` directory.

## How to Use the Investment Simulator

1. Navigate to the **Simulator** section of the website.
2. Input your investment parameters and click on the simulate button.
3. Review the results and explore different scenarios to understand investment dynamics.

Feel free to contribute to the project by submitting issues or pull requests! Happy investing!
